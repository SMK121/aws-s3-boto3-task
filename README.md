# ☁️ AWS S3 Storage Management Using AWS CLI and Python boto3

## 📌 Task Overview

This project demonstrates how Amazon S3 storage was managed using:

- AWS Command Line Interface (CLI)
- Python boto3 SDK

The task was completed using an Ubuntu 24.04 LTS EC2 instance.

The main objectives were:

- Configure AWS CLI and AWS authentication
- Create and manage S3 buckets
- Upload and download files from S3
- Delete S3 objects and buckets
- Automate S3 operations using Python boto3 scripts
